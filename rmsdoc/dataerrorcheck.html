    Check And Remove Data Error


    (1): On Front Panel on Menu (Top on the Panel), USER as Menu Item, Click Software Upgrade Open it, 
      Press Data Error Check button RMSSOFT display status, 
      If any Error seen then Press Remove Error button
      Repeat until all error not Removed.
  
  
        RMSSOFT के सबसे पहले पेज पर, सबसे उपर Menu बने हैं, उनमें से एक, पर USER लिखा है, उसे क्लिक करने पर नीचे की तरह कुछ और sub menu खुलेंगे, उनमें से एक पर लिखा है SOFTWARE UPDATE इसे खोलना है,
     वहां एक बटन Check Data Error लगा है इसे दबाने पर यदि कोई Error होगी तो दिखाई देगी, Error हटाने के लिए Remove Error का बटन दबाएं।
      यह प्रक्रिया तब तक दोहराएं जब-तक सभी Error हट न जाएं ।

        इसी USER menu में कई और भी विकल्प है जिसे आप अपने अनुसार RMSSOFT में बदलाव कर सकते हैं  ।
