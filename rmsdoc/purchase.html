    Purchase Panel


    (1): On Front Panel > Click or Press Enter on Purchase
      
    Purchase बटन पर Cursor ला कर Enter Key दबाएं,
    Purchase पैनल खुल जायेगा। 
  
    (2): Press Add Button >> This will Activate Sales Panel and Add Button will Disappear.
     
    Purchase पैनल खुलते ही ADD का बटन दबाना है पैनल  Activate/सक्रिय हो जाएगा, 
    अन्यथा पैनल Activate नहीं होगा और आप खरीद नहीं भर पाएंगे। 
    ADD बटन दबाते ही ADD बटन गायब हो जायेगा ।
 
    (3): Write Bill Dates or Use Current Dates >> Press Enter
     बिल Date वो Date है जिस तारीख को आप बिल कंप्यूटर/RMS में भर रहे हैं। 
     Invoice Date वो Date है जिस तारीख को Invoice/Bill Supplier द्वारा बनाया गया है। 
     इन्हे भरें और EnterKey दबाएं।  
 
    (4): Invoice Number Will Automatic in Sales 
     Invoice/Bill पर जो Invoice/Bill पर नंबर दिया गया है वही Invoice नंबर है, भरें और EnterKey दबाएं । 
     लेकिन Sales करते समय Invoice/Bill नंबर अपने आप बन जायेगा। 
 
    (5): Select CASH or CREDIT Bill >> Give Default Discount (will apply discount on 
     All items on in current bill ) >> Press Enter 
    (6): Write to Search Party Name >> Move Cursor UP/DOWN Key for selection >> Press Enter 
     अगर Party Name लिखने पर निचे बॉक्स में Party Name नहीं आ रहा है,
     मतलब Party का नाम अभी तक बनाया नहीं गया है,
     नया Party नाम बनाने के लिए, यहीं पर F1 key दबाएं 
     ( कई लैपटॉप पर Funtion Key, Fn के साथ F1 दबाने पर काम करती है ),
     नया पैनल खुल जायेगा, यहाँ पर Party की जानकारीया भरे Save करें Close करें । 
     अब दुबारा Party का नाम लिखें EnterKey दबाएं और आगे बढ़ें। 
 
    (7): Write to Search Item Name >> Move Cursor UP/DOWN Key for selection >> Press Enter 
     अगर Item Name लिखने पर बगल के बॉक्स में Item Name नहीं आ रहा है,
     मतलब Item अभी तक बनाया नहीं गया है,
     नया Item नाम बनाने के लिए, यहीं पर F1 key दबाएं 
     ( कई लैपटॉप पर Funtion Key, Fn के साथ F1 दबाने पर काम करती है ),
     नया पैनल खुल जायेगा, यहाँ पर Item की जानकारीया भरे Save करें Close करें । 
     अब दुबारा Item का नाम लिखें EnterKey दबाएं और आगे बढ़ें।
 
    (8): Cursor Will Move for STOCK Selection >> Select >> Press Enter  
    (9): Write Qty (Quantity) Press Enter >> Write Bonus(if any) Press Enter >> Write Rates
     (** Rate A means Price to Retailers, if you have frenchise like bussines) 
    (10): You can directly Edit NetRate (Net Rate is Calculation after TAX, BONUS, DISCOUNT) 
     Press Enter >> After NetRate Column Cursor will Move to NEXT Row, if Row is not visible then 
     RMS will create NEW ROW >> Search For NEXT ITEM ;  
    (11): If billing Item is Finished >> Press TAB key >> Cursor will move to bottom right ;
     if you want additional charges on bill then write here otherwise press TAB/Enter key >>
     Cursor will move to comment Field (Bottom Left, just above SAVE Button);
     Write comment/remark if any >> Press Enter >> Cursor Moves to SAVE Button >> 
     Press Enter and SAVE Bill >> After Saving Bill Print Message will appear (in Sale Only)
     Press ESC or 0 if you don't want to take PrintOut.
     
     अगर बिल Item की Entry पूरी हो जाये तो Enter Key दबाते हुवे नई खाली Row पर Cursor को आने दें,
     यहाँ पर TAB Key दबाएं, जिससे Cursor दाहिने नीचे तरफ पहुँच जायेगा, 
     यहाँ अगर कुछ Extra Charge Invoice पर दिए गए हैं तो भरें,
     अन्यथा Enter Key दबाएं और आगे बढ़ें,
     Cursor Comment पर पहुँच जायेगा, यहाँ आप Invoice Bill से सम्बंधित कमेंट लिख सकते हैं, 
     Enter Key दबाएं,
     Cursor Save बटन पर पहुँच जायेगा,  
     Enter Key दबाएं SAVE करें। 
     (SALE करते समय Print Page का विकल्प भी आएगा, जिसमे कितने 
     PrintPage चाहिए लिखने होंगे, 
     अगर PrintPage नहीं चाहिए तो 0, या Esc Key दबाएं। )  

     
    (12): अगर कोई Item कई अलग अलग Suppliers से ख़रीदा जाता है, 
      तो आप उस Item की तुलना कर सकतें हैं, 
      जिससे आप पता कर सकते हैं Item कौन सप्लायर कम rate या ज्यादा rate पर Item दे रहा है । 
      इस जानकारी को शुरू करने के लिए Item Compare जो पैनल के नीचे बीच में "Export PDF" बटन के ऊपर 
      Item Compare Check-Box है उसे ON कर दें। 
      इसके बाद जब भी कोई Item आप Search करते हुवे Qty पर जायेंगे तो नीचे 
      Suppliers के साथ Item का rate, bonus, discount भी दिखेगा 
      और आप पता कर सकतें है कौन सी rate कहाँ से मिल रही है।

    (a): When Cursor on Party Name: Press F1 Function key will Open Account Panel.
         You can create Accounts of Customer or Suppliers SAVE and CLOSE will automatically 
         back to current work.  
         [ नयी पार्टी बनाने के लिए : जब Cursor Party-Name पर हो, 
           तो F1 Key दबाएं (कई Laptop, पर Function key 'fn' के साथ F1 key दबाई जाती है)। 
           जिससे Customer/Supplier Account Opening पैनल खुल जायेगा 
           उसे भर कर SAVE करें Close करें पार्टी का नाम लिखें (नयी पार्टी का ) Enter के दबाते हुए आगे बढ़ें।  ]
    (b): When Cursor on Item Name (Purchase Only) : 
         Press F1 Function key will Open ItemEntry Panel.
         You can create Items here SAVE and Close will automatically 
         back to current work.  
         ** if recently Save Items not Shown in Item List then Press F5 Function key (Refresh)
         and search again will show recetly save items. 
         When Cursor on Item Name (Sales Only) : 
         Press F1 Function key will Open ItemEntry Panel.
         You can search for items about its description, rack or hsn >> Press ESC >> 
         Back to current work.
         [नया Item बनाने के लिए : जब Cursor Item-Name पर हो, 
          तो F1 Key दबाएं (कई Laptop पर Function key 'fn' के साथ F1, key, दबाई जाती है)। 
          जिससे Add/Modify Item पैनल खुल जायेगा 
          Form उसे भर कर SAVE करें Close करें पार्टी का नाम लिखें (नयी पार्टी का ) Enter के दबाते हुए आगे बढ़ें।   
          Cursor अपने आप Item-Name पर आजायेगा और नया बनाया Item दुबारा लिखें Select करें। 
          अगर नया बनाया Item लिखने पर Item लिस्ट में दिखायी दे तो F5 key दबाएं, 
          दुबारा Search करें अब Item दिखाई देगा। ]
    (c): When Cursor on Qty:
         Click on Radio Button 'Item Compare' on Bottom of Panel >> now Write Quantity >>
         Will Show Item comparison of Last 3 Purchases at right bottom side of panel  
    (d): When Cursor on any one of Rates, Bonus, Dis Press F8 Function Key will Show
         Last Billing of Selected Item on Previous Item Display List on Top Left. 
      
           
    मुख्य पैनल/Front Panel पर Search Purchase का बटन दिखेगा उसे दबाएं, Search Purchase पैनल खुलेगा,
    कितनी तारिख से - कितनी तारिख के बीच की खरीद बिल खोजनी है, वो दोनों तारिख (From Date - To Date) भरें,
    Enter Key दबाएं, पार्टी नाम भरें, नीचे पार्टी का नाम बिल नंबर, तारिख के साथ दिखेगा
    फोकस उस पर ले जा कर Enter key दबाएं । 
    खरीद बिल खुल जायेगा EDIT का बटन दबाएं (जब तक EDIT बटन नही दबेगा बिल EDIT नही होगा ), 
    बिल सही करें और अंत में UPDATE बटन दबाएं।   

    ** अगर पार्टी का नाम बिल नंबर, तारिख के साथ नहीं दिख रहा है तो : 
    1: दो गलत तारिख के बीच बिल खोजा जा रहा है। 
      (जीतनी तारिख से कितनी तारिख के बीच के खरीद बिल खोजनी है, उसे सही करें या तारिख बढ़ाएं या घटायें। )
    2: वह बिल बना ही नहीं है।  

    ** ध्यान रहे :- पुराने बिल में Supplier, का नाम नहीं बदला जा सकता।  

    


       
    अगर खरीद बिल Delete कर दिया जाये, तो उसका स्टॉक (बिल का स्टॉक) कम नहीं होता,
    उसको स्वयं ही डिलीट/कम/घटना होगा।  

    


         ** प्रत्येक Item की Qty Bonus Rate Discount Tax ठीक से दुबारा जांचे, 
     जो और जैसा खरीद बिल में दिया गया है वैसा ही भरा जाना चाहिए तो अंतर नहीं आएगा, 
     इनमे अंतर है तभी Total Invoice Value समान नहीं आ रहा (अंतर है),
     जबतक खरीद बिल का Total Invoice Value और rmssoft का Total Invoice Value 
     समान (एक जैसा) नहीं हो, कृपया खरीद बिल Save न करें।
     
     *** अगर सभी Item, सही भरने के बाद भी Total Value सही नहीं आ रही है तो, 
         Supplier द्वारा दी गयी Invoice गलत है।
