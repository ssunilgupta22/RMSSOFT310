    दूसरे Computer में RMSSOFT को ले जाने से पहले इसका BACKUP जरूर लेना होगा,
BACKUP FILE 'mrms.sql'  नाम से D:\RMSSOFT\win_rms\RMS_BACKUP\mrms.sql में रहती है, इसे सेफ लोकेशन पर रख लें, 
RMSSOFT को कॉपी कर लें और नए सिस्टम/कंप्यूटर में D:\ Drive या C:\ Drive के अलावा किसी और Drive में पेस्ट कर लें,
नए सिस्टम/कंप्यूटर में Paste किये गए RMSSOFT को OPEN करें 
इसमें mrms_pro.exe नाम की Application हैं, इसे Open करें  



क्यूंकि अभी नए System में SERVER Install नहीं है इसलिए RMSSOFT Open होने के बजाए SERVER Installation की तरफ जायेगा 
और आप को SERVER Install करना है SERVER Installation की गाइड भी PDF Document के साथ खुल जाएगी, 
इस PDF File को देख कर Installation करें, जैसे दिया गया है वैसे ही करें... ! FINISH करें । 
और अब दुबारा से mrms_pro (Application) को Open करें, अब RMSSOFT थोड़ा समय लेने के बाद खुल जायेगा 
लेकिन अभी RMSSOFT Blank होगा,
आपको अपना पुराना DATA इसमें RESTORE करने के लिए RESTORE DATA का बटन दबाना है और BACKUP DATA File mrms.sql File का Path Select करना है, 
फिर ओके बटन दबाना है
RMSSOFT DATA RESTORE करना शुरू कर देगा, wait.. करें... अब RMSSOFT को CLOSE कर के दुबारा OPEN करें आपका पुराना DATA वापस आ चूका होगा । 
अगर आपने BACKUP DATA File का PATH सही नहीं दिया है तो DATA RESTORE नहीं होगा 

*** BACKUP DATA FILE "CORRUPT" नहीं होनी चाहिए । इसका ध्यान रखना यूजर की जिम्मेदारी है ।
CORRUPT BACKUP FILE को RESTORE करने के बाद RMSSOFT ठीक से RUN न करे या OPEN ही न हो, 
इसलिये CORRUPT BACKUP FILE को RESTORE न करें ।
